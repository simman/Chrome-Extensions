WMTranslate For Chrome Extensions
============================
WMTranslate 是一款Chrome 浏览器的扩展，主要提供了基于有道API的翻译功能。


Features
--------

* 使用了`jQuery`
* 使用了 `localStorage` 进行本地存储已经查过的词语，方便下次直接使用。
* 支持设置是否自动发音，下一版实现按钮触发发音
* 提供了三种可选的提示窗口
