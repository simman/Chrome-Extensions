Chrome-Extensions
=================

Chrome Extensions By Me


WMTranslate
----------------------------------
WMTranslate 是一款Chrome 浏览器的扩展，主要提供了基于有道API的翻译功能。

WMQuick Response
----------------------------------
WMQuick Response 是一款Chrome 浏览器的扩展，主要提供快捷的二维码生成

WMShort URL
----------------------------------
WMShort URL 是一款Chrome 浏览器的扩展，主要提供快捷的短链接生成
